My NixOS, window-manager, Neovim, terminal, and tmux configuration, plus the scripts that connect them. The system is an ongoing effort to make navigating the shell and editing code feel more like muscle memory than active effort.

It includes a [tmux session manager](https://github.com/e-cal/dotfiles/blob/main/shared/scripts/tm) that has become central to my daily workflow.
