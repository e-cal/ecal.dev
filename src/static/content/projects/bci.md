A brain-computer-interface game controller built with the OpenBCI Ultracortex Mark IV using BrainFlow.

Signal-processing filters and transforms turn raw electrode readings into features for several machine-learning models. A simple SVM achieved the best validation performance at 94.4%.

Demonstrated at CUCAI 2023 by playing Flappy Bird in front of a live audience.
